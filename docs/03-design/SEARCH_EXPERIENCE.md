# Search Experience

> Search is one of the primary entry points into Plan My Stop. Unlike traditional search engines that return matching results, Plan My Stop understands intent, context, and preferences to deliver intelligent recommendations.

---

# Purpose

Users should never feel like they are searching through a database.

They should feel like they are asking an experienced travel expert.

The objective is not to find places.

The objective is to find the best experience.

---

# Nearby Search

Automatically prioritize

- Along Current Route
- Nearby Destination
- Walking Distance
- Driving Distance
- Journey Direction

Avoid recommending places behind the user unless requested.
